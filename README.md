## Installation & Usage

```sh
npm install color-picker-menu
OR
yarn add color-picker-menu
```

#### Include the Component

```js
import React from 'react';
import { ColorPickerMenu } from 'color-picker-menu';

const Component = () => {
  return <ColorPickerMenu />;
};
```
